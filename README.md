## Env

  - Ubuntu
  - meteor
      - create app `meteor create myapp`
      - install bootstrap `meteor add twbs:bootstrap`
  - npm
      - `npm install react react-dom --save`

## Run

  - `cd myapp && meteor`
      - server will listen at port 3000

## NOTE

  - js syntax: using ES6 rather than ES5 in most of this repo
